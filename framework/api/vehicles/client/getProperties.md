# getProperties

```lua
NCs.Vehicles:getProperties(vehicleId)
```

Get the properties of the specified vehicles.

### Arguments

| Argument  | Type   | Optional | Default | Explanation           |
| --------- | ------ | -------- | ------- | --------------------- |
| vehicleId | number | false    | -       | The ID of the vehicle |
